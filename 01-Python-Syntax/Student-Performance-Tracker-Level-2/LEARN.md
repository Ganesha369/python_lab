# Student-Performance-Tracker

[🚀 RUN LIVE](https://ganesha-dsa-hub.streamlit.app/?path=01-Python-Syntax/Student-Performance-Tracker-Level-2/main.py) | [💡 REQUEST](https://github.com/Ganesha369/python_lab/issues/new)

### 🏢 Real-World
In real-world education and HR settings, data is often stored as collections of information. Automating the calculation of averages, determining pass/fail status, and generating summaries saves hours of manual work and prevents human calculation errors. This project practices the core foundations of Python: lists, dictionaries, loops, and functions.

### 🛠️ Tech
Python 3 (Core Syntax: Lists, Dictionaries, Conditional Logic, and F-Strings).

### 📘 Guide
1. **Define Data Structures**: Create a list of dictionaries where each dictionary represents a student with their name and a list of test scores.
2. **Create a Logic Function**: Write a function that takes a list of numbers, calculates the average, and returns a letter grade based on specific thresholds (e.g., >90 is an A).
3. **Iterate Through Data**: Use a `for` loop to go through each student in your list.
4. **Apply Logic**: Inside the loop, call your function to process the scores.
5. **Format Output**: Use Python f-strings to print a clean, readable report showing the student's name, average score, and final grade.
